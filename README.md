## Task 04 - Sentiment Analysis

Performed sentiment analysis and visualization on Twitter social media data to understand public opinion patterns.

## Tasks Performed

* Loaded and cleaned Twitter dataset
* Renamed dataset columns
* Counted sentiment categories
* Visualized sentiment distribution using graphs

## Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Key Insights

* Negative sentiments were the highest
* Positive sentiments were also significant
* Neutral and irrelevant tweets were comparatively lower
